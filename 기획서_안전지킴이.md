# �️ 안전 지킴이 — 유아·여성 안전 특화 동네 안전 플랫폼

> **"우리 동네, 안전할까?"**
>
> 안전 지킴이는 시민이 직접 참여하여 동네의 안전 정보를 확인하고,
> **유아와 여성의 안전**을 지키기 위한 데이터 기반 안전 정보 웹서비스입니다.

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 안전 지킴이 |
| **한줄 소개** | 우리 동네 안전도를 한눈에 확인하고, 위험 구역 진입 시 지인에게 자동 알림을 보내는 유아·여성 안전 플랫폼 |
| **플랫폼** | 웹 (반응형, 모바일 대응) → 추후 PWA 확장 |
| **타겟 사용자** | 자녀를 둔 부모, 야간 귀가가 잦은 여성, 안전에 관심 있는 시민 |
| **핵심 가치** | 데이터 기반 안전 · 시민 참여 · 실시간 알림 |

---

## 2. 핵심 컨셉

### 🎯 문제 정의
- 우리 동네가 **안전한지 불안전한지** 한눈에 볼 수 있는 서비스가 없음
- 아이 등하원길, 여성 귀갓길의 **위험 요소(CCTV 사각지대, 어두운 골목)** 를 사전에 파악하기 어려움
- 위험 상황 발생 시 **지인에게 알리는 절차가 번거로움**
- 각종 안전 인프라(비상벨, 지킴이집 등) 위치를 **모르고 지나치는** 경우가 많음

### 💡 해결책
- 13개 공공 API 데이터를 지도 위에 **통합 시각화**
- 동네별 **안전 등급**을 한눈에 확인
- 위험 구역 진입 시 지인에게 **자동 푸시 알림**
- 시민이 직접 **위험 요소를 신고**하여 데이터를 보강

## 3. 주요 기능

### 3.1 🗺️ 동네 안전 지도 (메인 화면)
> 지도 위에 모든 안전 데이터를 레이어별로 표시하는 핵심 기능

| 레이어 | 데이터 소스 | 표시 정보 |
|--------|-----------|----------|
| 📹 **CCTV** | 전국 CCTV 표준데이터 | CCTV 위치, 대수, 설치 목적 |
| 💡 **가로등** | 가로등 위치 정보 | 가로등 밀도 (밤길 밝기) |
| 🔴 **비상벨** | 방범용 비상벨 위치 | 비상벨 위치 (탭하면 길찾기) |
| 🏠 **지킴이집** | 아동안전지킴이집 정보 | 위험 시 대피 가능한 장소 |
| 🚸 **스쿨존** | 전국 어린이보호구역 | 스쿨존 범위 표시 |
| 🌙 **안심귀갓길** | 여성 안심 귀갓길 데이터 | 검증된 안전 귀가 경로 |
| ⚠️ **주의 구간** | 아동범죄 주의 구간 + 범죄 통계 | 위험 지역 하이라이트 |
| 🚨 **시민 신고** | 사용자 직접 신고 데이터 | 시민이 올린 위험 정보 |

### 3.2 📊 동네 안전 등급
> 주소 검색 시 해당 동네의 안전 점수를 산출

| 평가 항목 | 산정 기준 | 비중 |
|----------|----------|------|
| CCTV 밀도 | 면적 대비 CCTV 수 | 25% |
| 가로등 밀도 | 면적 대비 가로등 수 | 15% |
| 비상벨·지킴이집 | 반경 내 안전 시설 수 | 20% |
| 범죄 발생률 | 지역별 범죄 통계 | 25% |
| 시민 신고 건수 | 최근 신고 빈도 | 15% |

**등급 체계:**
- ⭐⭐⭐⭐⭐ **S등급** — 매우 안전
- ⭐⭐⭐⭐ **A등급** — 안전
- ⭐⭐⭐ **B등급** — 보통
- ⭐⭐ **C등급** — 주의 필요
- ⭐ **D등급** — 위험

### 3.3 🚨 위험 신고
> 시민이 직접 위험 요소를 신고하여 안전 데이터를 보강

| 항목 | 설명 |
|------|------|
| 신고 카테고리 | 가로등 고장 / CCTV 사각지대 / 위험 시설물 / 수상한 사람·상황 / 기타 |
| 입력 정보 | 사진, 위치(GPS 자동 입력), 설명 텍스트 |
| **사진 관리** | **부적절한(성적/폭력적) 사진 업로드 방지를 위해 기능 제공**<br/>1. 유저 신고: 마커 팝업에 '부적절한 게시물 신고' 버튼 제공<br/>2. 블라인드: 누적 신고 일정 횟수(예: 3회) 도달 시 지도에서 자동 숨김 처리<br/>3. (향후) AI 비전 필터링 도입 고려 |
| 신고 상태 | 접수됨 → 확인 중 → 해결됨 |
| 공감 기능 | 다른 시민이 "나도 위험해요" 공감 버튼 |

### 3.4 🌙 안심 귀가 모드 (MVP 단순 버전)
> 버튼 하나로 지인에게 내 위치를 알리는 기능

| 단계 | 동작 |
|------|------|
| ① 시작 | 사용자가 **"안심 귀가 시작"** 버튼 탭 |
| ② 위치 공유 | 현재 위치 + 실시간 위치 공유 링크 생성 |
| ③ 알림 발송 | 등록된 지인에게 **푸시 알림** 발송: "🌙 [이름]님이 귀가 중입니다 📍[위치 링크]" |
| ④ 도착 | 목적지 도착 시 **"✅ 안전 도착"** 알림 자동 발송 |
| ⑤ 미도착 | 일정 시간 내 미도착 시 **"⚠️ 확인 필요"** 알림 발송 |

### 3.5 👶 아이 안전 모드
> 아이의 이동 경로에 있는 위험 요소를 사전에 체크

| 기능 | 설명 |
|------|------|
| 등하원길 안전 체크 | 출발지~도착지 경로상 CCTV, 지킴이집, 주의 구간 한눈에 확인 |
| 주변 안전 시설 | 반경 500m 내 비상벨, 지킴이집, 파출소 위치 표시 |
| 위험 구역 알림 | 아동 범죄 주의 구간 진입 시 보호자에게 알림 |

## 6. 디자인 방향

### 🎨 디자인 원칙
| 원칙 | 설명 |
|------|------|
| **신뢰감 있는 UI** | 안전 서비스답게 차분하고 신뢰감 있는 컬러 (블루 + 화이트 기반) |
| **직관적인 지도 중심** | 메인 화면 = 지도. 복잡한 메뉴 없이 바로 사용 가능 |
| **명확한 위험 표시** | 위험 구간은 빨강, 안전 시설은 초록/파랑으로 직관적 색 구분 |
| **모바일 퍼스트** | 이동 중 사용이 많으므로 모바일 사용성 최우선 |
